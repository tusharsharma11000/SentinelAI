# 🛡 SentinelAI

An AI-powered Border Surveillance System built using React, Node.js, MongoDB, Flask, OpenCV, and YOLOv8.

## Features

- Live Camera Feed
- AI Person Detection (In Progress)
- MongoDB Integration
- Modern Dashboard
- Alert Management
- Secure Backend APIs

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Python
- Flask
- OpenCV
- YOLOv8
